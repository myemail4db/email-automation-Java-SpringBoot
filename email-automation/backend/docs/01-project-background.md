# Project Background

The original version of the email automation workflow was first developed in Python to automate the manual process of organizing recruiter and job application emails.

The Python version focused on:
- Gmail API integration
- text cleanup
- document export
- batch email workflows

The Python email automation project reduced a manual 12-hour process down to about 90 minutes.

After validating the workflow and seeing how useful the automation process became, the project continued evolving into its next iteration. Here the Python project is rebuilt using Java Spring Boot to support a cleaner backend architecture and future expansion into workflow automation, analytics, and frontend integration.
